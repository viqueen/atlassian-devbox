## atlassian-devbox

### requirements

* [java - 8 / 11](https://adoptopenjdk.net/)
* [maven - 3.6.x / 3.8.x](https://maven.apache.org/install.html)
* [node](https://nodejs.org/en/)

### install it

```bash
git clone git@github.com:viqueen/atlassian-devbox.git
cd atlassian-devbox

npm install
npm link
```
